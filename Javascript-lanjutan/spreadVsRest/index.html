Spread operator dan Rest parameter

Spread operator ( ... )

Spread operator adalah memecah (expand/ unpack) iterables menjadi single element

object - object yang iterable :

  • String
  • Array
  • Arguments/ NodeList
  • TypedArray
  • Map
  • Set
  • User-defined Iterable

Rest parameter (...arg)

Rest parameter adalah merepresentasikan argument pada function dengan jumlah yang tidak terbatas menjadi sebuah array

Rest parameter hanya bisa diakhir argument
