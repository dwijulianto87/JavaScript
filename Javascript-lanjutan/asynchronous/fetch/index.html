Fetch

Fetch adalah sebuah method pada API javascript modern yang tugasnya mengambil data(resources) dari jaringan, dan mengembalikan promise yang selesai(fullfilled) ketika ada respon yang tersedia secara asynchronous atau ajax (XMLHttpRequest)

fetch(resources, init);

    resources

  • Url alamat dari sumber yang kita akan ambil
  • Request Object representasi sumber

    init (optional)

    konfigurasi tambahan pada sebuah request berbentuk object
