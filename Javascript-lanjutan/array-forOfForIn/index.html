for..of VS for..in

For..of

creates a loop iterating over iterable object

for..of adalah sebuah looping yang bisa mengulang atau menelusuri object - object yang iterable

object - object yang iterable :

  • String
  • Array
  • Arguments/ NodeList
  • TypedArray
  • Map
  • Set
  • User-defined Iterable

For..in

creates a loop only iterating over enumerable

For..in adalah sebuah looping yang bisa mengulang atau menelusuri hanya untuk enumerable (properti pada object
